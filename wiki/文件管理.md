## 打包、压缩
调用gzip
```
tar -czf aa.tar.gz /aa
```
调用bzip2
```
tar -cjf aa.tar.bz2 /aa
```
调用xz
```
tar -cJf aa.tar.xz /aa
```
解压、解包
```
tar -xvf aa.tar.gz -C /tmp
//-C解压到哪里
```
解压zip
```
unzip xxx.zip
```
