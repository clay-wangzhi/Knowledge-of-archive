# yum管理RPM包

清空缓存及其它文件

yum clean all
1

建立缓存数据库

yum makecache
1

查询可用的仓库

yum repolist
1

查询

yum list mysql-server
1

升级

yum -y update
1

查询指定的文件属于哪个包

yum provides */httpd
1

卸载

yum -y remove mysql-server
1

# 使用RPM工具管理RPM包

安装

rpm -ivh xxx.rpm
1

升级

rpm -Uvh xxx.rpm
1

查询

rpm -q xxx //查询指定包是否安装
rpm -ql xxx //查询xxx安装的文件
1
2

卸载

rpm -e xxx
1
