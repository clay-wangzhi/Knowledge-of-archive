# 2.3 Python数据类型

## 内建常用数据类型

* 数值型
  * int、float、complex、bool
* 序列 sequence
  * 字符串 str、字节序列 bytes、bytearray
  * 列表 list、元组 tuple

* 键值对
  * 集合 set、字典 dict

## 类型转换

* int、float、complex、bool 也可以当做内建函数对数据进行类型转换
* int(x) 返回一个整数
* float(x) 返回一个浮点数
* complex(x)、complex(x,y) 返回一个复数
* bool(x) 返回布尔值，前面讲过False等价的对象
